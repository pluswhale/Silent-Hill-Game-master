Readme

Silent Hill это игра на выживание, основанная на игре Silent Hill. Игра ужасов начинается с того, что игрок находится в одной комнате, где есть зелья, золото, боеприпасы и двери, ведущие в другие комнаты. В каждой волне монстры будут выходить и пытаться убить игрока. Главная цель игрока - выжить, сохраняя здоровье и здравомыслие, одновременно ища боеприпасы для стрельбы по монстрам и золото для строительства стен.

Язык программирования моей программы - Python 3. Основным используемым модулем является Pygame, который не предустановлен в python.

Чтобы установить на Mac, перейдите по этой ссылке:
http://florian-berger.de/en/articles/installing-pygame-for-python-3-on-os-x/
Чтобы установить на Windows, перейдите по этой ссылке:
https://docs.google.com/document/d/1A-FudgG9rdL-qcuGavGJ5Z9L7fW_nUQg9WuL6CwTs_Y/edit
Пакеты Pygame доступны для загрузки на веб-сайте Pygame:
http://www.pygame.org/download.html

Другие используемые модули включают случайный, математический и дата,время. Они предварительно установлены в python.
Чтобы запустить игру, откройте Game.py в папке "Исходные файлы" и "Файлы поддержки" вы запускаете файл python (Ctrl + B в Windows и Command + B на Mac).
